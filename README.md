# NASA-Climate-Story


Welcome to Climate Story, a website dedicated to exploring and analyzing climate data from various space agencies. The platform provides insights into global climate patterns and their impact on the environment.

## Features

- **Dataset Exploration**: Visualizations and insights from global space agency climate data.
- **Responsive Design**: Accessible across devices for easy navigation and readability.
- **Educational Content**: Learn about climate trends and environmental changes.

## Technologies Used

- **HTML/CSS**: For website structure and design.
- **Vercel**: Hosting platform.
